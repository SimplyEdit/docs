(Finally) A really simple website editor that just works.

The first inline WYSIWYG editor you can simply use anywhere, any time. It works with any CMS, such as Wordpress, Drupal. It works with Magento. It even works without a CMS or database. Simply Edit Anything.

SimplyEdit is an inline wysiwyg editor. It is easy to integrate and easy to extend.
